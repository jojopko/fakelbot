FAKEL bot
Бот-ретранслятор для отправки постов из группы ВК в telegram канал.

### Поддреживаемые события ###
 - Подтверждение (тех.)
 - Записи на стене: Добавление


### Сборка / Установка ###
Нужные программы:
 - Python 3.9.x или более новые.
 - pip
 - (*) Docker (при разработке использовался version 20.10.22)

Без Docker:
    1. Распаковать/клонировать проект и перейти в него
    2. Установить зависимости
       `pip install -r req.txt`
    3. Создать файл с конфирурацией ".env"
    4. Записать в файл ".env" следующие строки:
       ``
        SERVER_HOST=         # Хост сервера (на проде 0.0.0.0)
        SERVER_PORT=         # Порт приложения
        APP_SECRET=          # Секретный ключ из запроссов от ВК
        VK_CONFIRMATION=     # Ключ, который отправляется при авторизации сервера (смотреть в группе, вкладка "работа с API")
        VK_GROUP_ID=         # ID группы ВК
        TG_CHANNEL_NAME=     # Имя telegram канала без "@"
        TG_BOT_TOKEN=        # Токен бота, смотреть у @BotFather
       ``
    5. Записать нужные значения для каждой переменной
    6. Запустить программу
       `python web.py`
    7. Открываем веб-интерфейс ВК группы, чтобы отправить запрос на 
подтверждение. Группа->Управление->Работа с API->Callback API->Настройки сервера.
    8. В поле "Адрес" вписываем адрес сервера и дописываем "/vk".
    9. Прописываем секретный код как в конфирурации.
    10. Проверяем строку, которую должен вернуть серверу (тк. она меняется
со временем) и нажимаем "Подтвердить".
    11. В случае успеха (зеленая галка, напротив адреса) - переходим в
"Типы событий" и выбираем нужные (из поддерживаемых); В случае неудачи -
перепроверьте прошлые пункты, состояние вашего сервера и сервера ВК.

Установка с помощью Docker:
    1. Распаковать/клонировать проект и перейти в него
    2. Собрать docker image
       `docker build -t fakel/fakelbot:v0.1 .`
    3. Подготовить файл конфирурации как в иструкции выше
    4. Запустить контейнер. Однако надо правильно указать порты!
       `docker run -d -p 80:80 --env-file .env fakel/fakelbot`
    5. Подтвердить сервер как в инструкции выше
