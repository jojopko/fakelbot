FAKEL bot
Бот-ретранслятор для отправки постов из группы ВК в telegram канал.

### Поддреживаемые события ###
 - Подтверждение (тех.)
 - Записи на стене: Добавление


### Сборка / Установка ###
Нужные программы:
 - Python 3.9.x или более новые.
 - pip
 - (*) Docker (при разработке использовался version 20.10.22)

Без Docker:
    1. Распаковать/клонировать проект и перейти в него
    2. Установить зависимости
       `pip install -r req.txt`
    3. Создать файл с конфирурацией ".env"
    4. Записать в файл переименовать ".env.example" в ".env"
    5. Записать нужные значения для каждой переменной (Важно учитывать, что ВК оправляет только на 80 порт)
    6. Запустить программу (лучше через gunicorn или иные WSGI серверы)
       `python web.py`
    7. Открываем веб-интерфейс ВК группы, чтобы отправить запрос на подтверждение. Группа->Управление->Работа с API->Callback API->Настройки сервера.
    8. В поле "Адрес" вписываем адрес сервера.
    9. Прописываем секретный код как в конфирурации.
    10. Проверяем строку, которую должен вернуть серверу (тк. она меняется со временем) и нажимаем "Подтвердить".
    11. В случае успеха (зеленая галка, напротив адреса) - переходим в "Типы событий" и выбираем нужные (из поддерживаемых); В случае неудачи - перепроверьте прошлые пункты, состояние вашего сервера и сервера ВК.

Установка с помощью Docker и nginx:
    1. Распаковать/клонировать проект и перейти в него
    2. Собрать docker image
       `docker build -t fakel/fakelbot:latest .`
    3. Подготовить файл конфирурации как в иструкции выше
    4. Настроить reverse proxy в nginx (файл /etc/nginx/sites-available/default) как в примере "default.example"
    5. Запустить контейнер. Однако надо правильно указать порты!
       `docker run -d -p 81:5000 --env-file .env fakel/fakelbot`
    6. Подтвердить сервер как в инструкции выше
